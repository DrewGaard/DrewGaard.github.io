TrackHub

TrackHub is a web app built using nodejs along with an SQL database. It allows users to browse and listen to songs using the Napster API.

TrackHub is an interactive music player that allows users to create an account and stream music. One of our goals when creating TrackHub was to make it cross platform, so we implemented it as a website available to run in any modern web browser. TrackHub aims to be a hub for music lovers and enthusiasts to listen, share, chat, and enjoy their music experience.
