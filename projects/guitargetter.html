GuitarGetter

GuitarGetter was created to allow users to enter information about guitarists gear into a database.

GuitarGetter was created as a project for my databases class at Pacific. The database was originally implemented with phpmyadmin and MySQL, but was later migrated to Postgres on Heroku so that the website and database could be accessed online. GuitarGetter allows users to enter information into the database, retrieve information from the database, and view the database.
