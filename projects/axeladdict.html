AxelAddict

AxelAddict is a Java application designed to match users with cars that fit their personality. It also features search abilities with API integration and 0-60/quarter mile calculators.

Axle Addict is an application created using java with the purpose of matching a users personality with a car. The application also features a 0-60 calculator, quarter mile calculator, and a car search engine. I worked with a group of 5 other colleagues to create the application. Together, in under 3 months we were able to create a working application that included all the basic features we outlined at the start of the semester. We used Scrum to stay on track. The project utilizes the Google search API and the Edmund's car API.
